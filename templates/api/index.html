Django Books API

Django Books Store API can be access to get JSON format Book data

Just click the button and enter the ISBN of the particular book from the book detail or search for the book and enter its url to get JSON formatted data of the book.

Get JSON Data

Get Started

Get JSON formatted data of book containing the book title, author, year, review count and average score in the format given below
{
"title": "Memory",
"author": "Doug Lloyd",
"year": 2015,
"isbn": "1632168146",
"review_count": 28,
"average_score": 5.0
}

Using The ISBN Form

  1. Enter the ISBN of the book in the form and click 'GET'
  2. A new window will open with the JSON data for the book
  3. There is no limit on requesting for JSON data

Using The ISBN Url

  1. Enter in the browser 127.0.0.1/api/isbn
  2. Copy the JSON displayed
  3. Unlimited request
  4. Json data using python
  5. import requests

    data = requests.get('http://127.0.0.1:8000/api/0778325504')
    print(data.json())

    json = data.json()
    print(json['title'])
  6. That's it.
